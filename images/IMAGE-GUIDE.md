# Product Images Setup Guide

## Current Status
The website is configured to use local product images, but no images have been added yet.

## What You Need to Do

### Option 1: Use the Placeholder Generator
1. Open `generate-placeholders.html` in your browser
2. Click "Download All Images" to get test placeholder images
3. Save all downloaded images to this `images` folder
4. Replace them with your actual product photos when ready

### Option 2: Add Your Own Images
1. Take or obtain photos of each product
2. Name them exactly as listed below
3. Place them in this `images` folder
4. Recommended: 400x400 pixels, JPG format, under 200KB each

## Image Tips
- Use good lighting
- Show the product clearly
- Use a neutral or white background
- Keep file sizes reasonable (compress if needed)
- Square format (1:1 aspect ratio) works best

## Testing
Once images are added, open `index.html` in your browser to see them displayed on the website.
