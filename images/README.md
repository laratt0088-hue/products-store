## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 400x400 pixels (square format)
- **Aspect Ratio**: 1:1 (square) for best display
- **File Size**: Optimize images to keep file sizes reasonable (under 200KB recommended)

## Adding Images

1. Place your product images in this folder
2. Name them exactly as shown above (product-1.jpg, product-2.jpg, etc.)
3. Ensure images are clear, well-lit, and show the product prominently
4. The website will automatically display these images

## Notes

- If an image is missing, a placeholder will be displayed
- Images are referenced in `script.js` with the path `images/product-[id].jpg`
- Each product has an Arabic name comment in the code for easy identification
