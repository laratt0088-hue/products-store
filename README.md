# Online Store Project

This project is an online store where users can select products and add them to their favorites page.

## Key Features
- Select products and add them to the favorites page.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the `index.html` file in your browser.
2. Browse the products and add your favorite items to the favorites page.
3. Choose your rating for the website on the comments page.
